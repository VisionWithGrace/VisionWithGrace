Elements of our application are heavily dependent on precompiled binaries. As a result, the executable and necessary binaries are too large to be uploaded to CTools.

The repository may be accessed at: https://github.com/VisionWithGrace/VisionWithGrace
Please see the README.md in the top folder of the repo for further installation and execution instructions.

To run the executable, navigate to: 
VisionWithGrace\VisionWithGrace\bin\Release\VisionWithGrace.exe
Double-click VisionWithGrace.exe to run the application.
